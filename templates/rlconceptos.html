Conceptos básicos de Regresión Lineal

La regresión lineal es una de las herramientas estadísticas más utilizadas para analizar la relación entre variables. Su propósito es modelar cómo una variable dependiente se comporta en función de una o más variables independientes, lo que permite tanto describir tendencias como hacer predicciones (Dagnino, 2014). Por ejemplo, en un contexto educativo, podría usarse para estimar la calificación final de un estudiante a partir de sus horas de estudio y asistencia a clase.

Cuando se extiende a más de una variable explicativa, se habla de regresión lineal múltiple, la cual proporciona un panorama más completo en situaciones donde intervienen varios factores al mismo tiempo. Este enfoque se estima generalmente mediante mínimos cuadrados ordinarios, buscando que el modelo sea insesgado y eficiente bajo los supuestos clásicos del modelo lineal (Uriel, 2013).

  • Linealidad: la relación entre las variables debe ser lineal.
  • Independencia: los errores deben ser independientes.
  • Homoscedasticidad: la varianza de los errores debe ser constante.
  • Normalidad del error: los residuos deben seguir una distribución normal.

Un dataset de entrenamiento es el conjunto de datos utilizado para que un modelo aprenda patrones y relaciones entre variables. En el caso de la regresión lineal, contiene valores de entrada (variables independientes) y de salida (variable dependiente), lo que permite al algoritmo ajustar la recta o el hiperplano que mejor represente esas relaciones.

Ejemplos de datasets muy utilizados en aprendizaje automático son:
  • Boston Housing Dataset: precios de viviendas en Boston en función de variables socioeconómicas y ambientales (Harrison & Rubinfeld, 1978).
  • Iris Dataset: características morfológicas de flores, muy usado en clasificación pero adaptable a regresión (Fisher, 1936).
  • Wine Quality Dataset: propiedades físico-químicas de vinos para predecir su calidad (Cortez et al., 2009).

    • Un dataset posee varias propiedades importantes como el número de filas (observaciones) y columnas (características), los tipos de datos que pueden ser numéricos, categóricos o de texto, la presencia de valores faltantes que deben limpiarse, posibles duplicados que afectan la calidad, así como estadísticas básicas (media, mediana, desviación estándar, etc.) y, en caso de problemas de clasificación, el balance de clases que muestra la distribución de las categorías en la variable objetivo.

Estos datasets permiten practicar conceptos de regresión lineal y son la base para evaluar la capacidad predictiva de un modelo.

Imaginemos que queremos predecir la nota final de un estudiante en función de las horas de estudio.

El modelo de regresión lineal ajusta la recta:
Y = 50 + 5X
Eso significa que cada hora de estudio aumenta en promedio 5 puntos la nota del estudiante.

Ejemplo de predicción: Si un estudiante estudia 6 horas, el modelo estima:
Y = 50 + 5(6) = 80

Supongamos que tenemos 10 datos de horas de estudio y notas. Para entrenar un modelo, los dividimos así:
  • Training set (70%): 7 datos para que el modelo aprenda.
  • Validation set (15%): 1–2 datos para ajustar parámetros.
  • Test set (15%): 1–2 datos para evaluar el modelo con ejemplos nuevos.

Con esta división, el modelo aprende con los primeros 7 datos, se ajusta con los de validación y se prueba con los últimos.

Cumplir con estos supuestos garantiza que las estimaciones del modelo sean confiables y que las inferencias estadísticas tengan validez. En caso contrario, las conclusiones podrían ser engañosas y sería necesario considerar transformaciones o modelos alternativos (Nabi, 2021; Uriel, 2013).
